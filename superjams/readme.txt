This distribution of JAMS contains the J2K hydrological model and
a test dataset. You can quickly start as follows:
1. Start JAMS Builder (JUICE)
2. From the menu, choose File -> Load Model
3. From the data/j2k_gehlberg directory, select the j2k_gehlberg.jam
   model configuration file
4. Apply settings to initial model parameters as you like
5. Run the model and enjoy ;)
